Come si usa git?

Tutti i comandi git iniziano con git (ma dai...)
seguito dall'operazione che vogliamo compiere

git {comando} [{sotto_comando}]
Iniziamo

Devo fare altro per usare git?

Git per funzionare richiede che almeno
i parametri user.name e user.email
siano configurati sulla nostra macchina

Per far questo usiamo il sotto-comando config

git config \-\-global user.name il_mio_nome
git config \-\-global user.email mia_mail@poul.org

Siamo pronti per usare git

N.B. Il parametro \-\-global impone che l'impostazione sia configurata
per l'utente corrente, solamente sul nostro computer
