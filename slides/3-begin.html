Let's do this

How to install git?

Ubuntu/Debian
apt install git
Arch/Majaro
pacman -S git
Fedora
dnf install git

For the other systems there is the official guide

How to use git?

All git commands begin with git (seriously?!)
followed by a command

git <command> [<subcommand>]

Do we need anything else to use git?

To work, git needs that at least the parameters
user.name and user.email are set

To do this, we use the command config

git config --global user.name <my_name>
git config --global user.email <my_email>

We are ready to use git

n.b. the argument --global implies that the configuration
is about current user, only on our local machine
