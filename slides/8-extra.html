Submodule

git allows to put a repository into
another one as a module of it

e.g. this allows to separate a project in some repositories,
everyone with its independent history

Submodule on git-scm.com
Extra

Tagging

It's possible to tag the commits

e.g. this feature is often used to annotate
project version at a specific commit

Tagging on git-scm.com
Extra

GPG signing

git has built-in function to signing
commits with a GPG signature

This permits to certificate the integrity and origin of commit

Signing on git-scm.com
Extra

Hosting self-hosted

If you don't trust public hosting of repositories as
gitlab.com or github.com,
you can also create your own server of git

Compared to the more famous gitlab,
it is advisable gitea, clean and light
