Submodule

Git permette di includere un repository dentro
un altro come sotto modulo

Questo permette per esempio di separare un progetto
in piú repository con la propria storia indipendente

Submodule on git-scm.com
Extra

Firma GPG dei commit

Git predispone un sistema automatico di firma dei commit

Questo permette di certificare la provenienza dei commit

Signing on git-scm.com
Extra

Hosting self-hosted

Se non ti fidi degli hosting remoti come
gitlab.com o github.com puoi sempre hostare
un servizio di hosting git su una tua macchina

Rispetto al piu famoso gitlab é consigliabile
gitea, pulito e leggero
